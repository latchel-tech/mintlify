description: "To download and activate the **Latchel Data Sync** extention, it's important to follow these steps in sequence:"
mode: "wide"
---

1. First, go through the help guide below to **download and install** the Latchel Data Sync Extension.

<Card title="How to Install the Latchel Data Sync Extension" icon="plug-circle-check" iconType="duotone" href="/s/topic/0TO5e000000h9wqGAA/How-to-Install-the-Latchel-Data-Sync-Extension">
  Follow these steps to download and install the Latchel Data Sync Extension.
</Card>

- Once signed into the extension successfully, you will see text on the extension drop-down that states, **"Logged in as PM Company Name"**.

2. Create a **fake work order** in your Property Management Software (PMS) and **assign this fake work order to the Team Latchel user**.

<Note>
  Please be sure to assign Team Latchel as a user, **NOT** a service provider.
</Note>

3. Sign out of the Latchel Data Sync extension, and then sign back in.
4. Sign out of your property management software and then sign back in.
5. Navigate back to your property management software dashboard where you should see a floating Latchel logo at the top right.

   <Frame>
     ![Latchel Data Sync Extension Pn](/images/LatchelDataSyncExtension.png)
   </Frame>
   **To know more about navigating the new extension, visit:**

   <Card title="Navigating the Latchel Data Sync Extension" icon="plug-circle-check" iconType="duotone" href="/s/topic/0TO5e000000h9wqGAA/Navigating-the-Latchel-Data-Sync-Extension-New-Version">
     Explore the layout and key functions of the latest Latchel Data Sync Extension.
   </Card>
6. **Auto-sync is now always on by default** with the new Latchel Data Sync Extension, **refreshing data every 4 hours.**
7. Once the sync has been initialized, you can **complete** or **close** the test work order in your property management software (PMS).
