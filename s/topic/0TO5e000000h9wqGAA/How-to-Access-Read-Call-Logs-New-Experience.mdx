From the dashboard, click on **Maintenance**, and open the work order by either **searching and/or clicking the specific order number**. You can also use the **Filter** and **Clear Filters** option for a more refined search. 

<Tip>
  To learn more about how to use the **Filter**, check out this help guide: [**How to Use the Filter Options in the Work Order Tab Page**](https://help.latchel.com/s/topic/0TO5e000000h9wqGAA/How-to-Use-the-Filter-Options-in-the-Work-Order-Tab-Page)**.**
</Tip>
