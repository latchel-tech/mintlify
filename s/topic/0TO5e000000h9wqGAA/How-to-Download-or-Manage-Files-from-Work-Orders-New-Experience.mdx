---
title: How to Download or Manage Files from Work Orders (New Experience)
description: "Keep track of pictures, invoices, receipts, and other files from a work order. Here's how you can download or manage files from a work order."
mode: wide
---

From your Dashboard, click the **Maintenance** dropdown and select **Work Orders**.

<Frame>
  <img src="/images/rtaImage-400.jpg"/>
</Frame>

Now to select the desired work order, use the **Search Bar** then click on the Work Order ID on the Work Order page. You can also use **Filter** for easy search. 


<Frame>
  <img src="/images/rtaImage-401.jpg"/>
</Frame>

From the work order page, go to the **Files** tab, the saved file/s can be previewed before downloading or deleting. 

<Frame>
  <img src="/images/rtaImage-402.jpg"/>
</Frame>

To preview the file, click on the **File Name** and a new window will appear. You may view if it’s the right file you wish to manage. 

<Frame>
  <img src="/images/rtaImage-403.jpg"/>
</Frame>

Sample Preview File: Invoice


<Frame>
  <img src="/images/rtaImage-404.jpg"/>
</Frame>

Once you have confirmed it's the file you need to manage, you can either download a file by clicking the **download icon** or delete it, when necessary, by clicking the **trash icon**.

<Frame>
  <img src="/images/rtaImage-405.jpg"/>
</Frame>

<Info>
**NOTE:**

* _All downloaded files are saved in the downloads folder of the Property Manager or Latchel Admin user's computer._
* _Deleting a file is _irreversible.__
* _Different types of files can be saved and previewed such as (.)png, (.)pdf, movie or a gif, and docx._

</Info>
