---
title: "How to Create User Groups"
description: "User Groups define a group that will receive emails when a job alert is triggered. You can create one or multiple groups wherein each group may have members belonging to another group. This is to ensure that appropriate notifications are received and the intended recipients can respond immediately. You can either edit the existing Default Notification Group or create new groups."
---

### **Configuring a User Group (Company Level)**

<Warning>
  The User Group at the Company Level will automatically apply to all portfolios and properties on the account unless there are individual assignments made at the property or portfolio level. \
  \
  To learn more about configuration inheritance and how to override company-level configurations, see [Configuration Inheritance (Property, Portfolio, & Company Level).](/s/topic/0TO5e000000h9wqGAA/Configuration-Inheritance-Property-Portfolio-Company-Level)
</Warning>

Click on the **drop-down** at the top right-hand side of your screen. Select **Account Settings** from the menu.

<Frame>
  ![1latchel Pn](/images/1Latchel.png)
</Frame>

From the Account Settings page, click the **User Groups** tab.

<Frame>
  ![](/images/rtaImage-445.jpg)
</Frame>

On the right-hand side of your page, click **New User Group.**

<Frame>
  ![](/images/rtaImage-446.jpg)
</Frame>

Type in a name for your User Group and click**Save.**

<Frame>
  ![](/images/rtaImage-447.jpg)
</Frame>

On the far right-hand side of your screen, click the **ellipsis (**<Icon icon="ellipsis" color="#000000" />**)** and then **Add User to Group.**

<Frame>
  ![](/images/rtaImage-448.jpg)
</Frame>

Search for an existing user on your account or create a new user. Click **Submit** to save.

<Frame>
  ![](/images/rtaImage-449.jpg)
</Frame>

To set the User Group to receive all of Latchel’s email notifications on the Company Level, click the **Details** tab.

Under the **Maint. Communication Settings**, click the dropdown under the **Default Notification User Group** field and select the User Group. Click **Save.**

If you don’t see the User Groups you need to refresh the page so they get populated.

<Frame>
  ![](/images/rtaImage-450.jpg)
</Frame>

### **Configuring a User Group (Portfolio Level)**

<Note>
  The User Group at the Portfolio Level will automatically be applied to all properties within that portfolio. \
  \
  To learn more about configuration inheritance and how a single property can override Portfolio and company-level configurations, see [Configuration Inheritance (Property, Portfolio, & Company Level)](/s/topic/0TO5e000000h9wqGAA/Configuration-Inheritance-Property-Portfolio-Company-Level).
</Note>

Click the **Places** tab and then, click **Portfolios**. Then, search or select the portfolio that you want to edit.

<Frame>
  ![Latchel 07 02 2025 07 47 AM Pn](/images/Latchel-07-02-2025_07_47_AM.png)
</Frame>

<Frame>
  ![Latchel 07 02 2025 07 50 AM Pn](/images/Latchel-07-02-2025_07_50_AM.png)
</Frame>

On the **Details** page, scroll down to the **Maint. Communication Settings** section.

<Frame>
  ![](/images/rtaImage-453.jpg)
</Frame>

Once you expand the **Maint. Communication Settings**, click **Override** under the **Default Notification User Group**.

<Frame>
  ![](/images/rtaImage-454.jpg)
</Frame>

From here, you will see the Company Level Notification User Group was automatically applied to the portfolio. To override the settings,  select the User Group from the dropdown under the **Default Notification User Group** field. Click **Save.**

<Frame>
  ![](/images/rtaImage-455.jpg)
</Frame>

### **Configuring a User Group (Property Level)**

Click the **Places** tab and then, click **Properties**.

<Frame>
  ![Latchel 07 02 2025 07 26 AM Pn](/images/Latchel-07-02-2025_07_26_AM.png)
</Frame>

Search or select the property **address.**

<Frame>
  ![Latchel 07 02 2025 07 29 AM Pn](/images/Latchel-07-02-2025_07_29_AM.png)
</Frame>

On the **Details** page, scroll down to the **Maint. Communication Settings**. From here, you will see the property inherited from the Company Level User Group. To override this, click **Override** under **Default Notification User Group**.

<Frame>
  ![](/images/rtaImage-458.jpg)
</Frame>

Then select the User Group dropdown under the **Default Notification User Group** field. Click **Save.**

<Frame>
  ![](/images/rtaImage-459.jpg)
</Frame>
