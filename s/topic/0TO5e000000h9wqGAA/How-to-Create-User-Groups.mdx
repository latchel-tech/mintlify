---
title: How to Create User Groups
description: "User Groups define a group that will receive emails when a job alert is triggered. You can create one or multiple groups wherein each group may have members belonging to another group."
mode: wide
---


This is to ensure that appropriate notifications are received and the intended recipients can respond immediately. You can either edit the existing Default Notification Group or create new groups.

#### **Configuring a User Group (Company Level)**


<Info>
  **NOTE:** _The User Group at the Company Level will automatically apply to **all portfolios and properties** on the account. To learn more about configuration inheritance and how to override company-level configurations, click **[here](/s/topic/0TO5e000000h9wqGAA/Configuration-Inheritance-Property-Portfolio-Company-Level).**_
</Info>

Click on the **drop-down** at the top right-hand side of your screen. Select **Account Settings** from the menu.

<Frame>
  <img src="/images/rtaImage-444.jpg"/>
</Frame>


From the Account Settings page, click the **User Groups** tab.

<Frame>
  <img src="/images/rtaImage-445.jpg"/>
</Frame>


On the right-hand side of your page, click **New User Group.**


<Frame>
  <img src="/images/rtaImage-446.jpg"/>
</Frame>



Type in a name for your User Group and click**Save.**


<Frame>
  <img src="/images/rtaImage-447.jpg"/>
</Frame>


On the far right-hand side of your screen, click the **three horizontal dots** and then **Add User to Group.**

<Frame>
  <img src="/images/rtaImage-448.jpg"/>
</Frame>


Search for an existing user on your account or create a new user. Click **Submit** to save.

<Frame>
  <img src="/images/rtaImage-449.jpg"/>
</Frame>



To set the User Group to receive all of Latchel’s email notifications on the Company Level, click the **Details** tab. Under the **Maint. Communication Settings**, click the dropdown under the **Default Notification User Group** field and select the User Group. Click **Save.** If you don’t see the User Groups you need to refresh the page so they get populated.


<Frame>
  <img src="/images/rtaImage-450.jpg"/>
</Frame>


#### **Configuring a User Group (Portfolio Level)**

**NOTE:**_The User Group at the Portfolio Level will automatically be applied to **all properties within** that portfolio. To learn more about configuration inheritance and how a single property can override Portfolio and company-level configurations, click **[here](/s/topic/0TO5e000000h9wqGAA/Configuration-Inheritance-Property-Portfolio-Company-Level).**_

Click the **Places** tab and then, click **Portfolios**. Then, search or select the portfolio that you want to edit.


<Frame>
  <img src="/images/rtaImage-451.jpg"/>
</Frame>

<Frame>
  <img src="/images/rtaImage-452.jpg"/>
</Frame>


On the **Details** page, scroll down to the **Maint. Communication Settings** section.

<Frame>
  <img src="/images/rtaImage-453.jpg"/>
</Frame>


Once you expand the Maint. Communication Settings**,** click **Override** under the **Default Notification User Group**.

<Frame>
  <img src="/images/rtaImage-454.jpg"/>
</Frame>


From here, you will see the Company Level Notification User Group was automatically applied to the portfolio. To override the settings,  select the User Group from the dropdown under the **Default Notification User Group** field. Click **Save.**

<Frame>
  <img src="/images/rtaImage-455.jpg"/>
</Frame>

#### **Configuring a User Group (Property Level)**

Click the **Places** tab and then, click **Properties**.

<Frame>
  <img src="/images/rtaImage-456.jpg"/>
</Frame>



Search or select the property **address.**

<Frame>
  <img src="/images/rtaImage-457.jpg"/>
</Frame>


On the **Details** page, scroll down to the **Maint. Communication Settings**. From here, you will see the property inherited from the Company Level User Group. To override this, click **Override **under **Default Notification User Group**.

<Frame>
  <img src="/images/rtaImage-458.jpg"/>
</Frame>



Then select the User Group dropdown under the **Default Notification User Group** field. Click **Save.**


<Frame>
  <img src="/images/rtaImage-459.jpg"/>
</Frame>
