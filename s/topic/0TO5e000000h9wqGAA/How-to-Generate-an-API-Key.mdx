---

Latchel Property Managers can generate (for the first-time) or regenerate an API key right from the LatchelÂ [console](http://app.latchel.com/)\! Keep reading to learn more.

### **How to generate an API Key for the first time?**

On your Latchel dashboard click the initial icon and on the dropdown select **Account Settings.**

<Frame>
  ![1latchel Pn](/images/1Latchel.png)
</Frame>

Select the **Integrations** tab from the sub-navigation menu.

<Frame>
  ![](/images/rtaImage-819.jpg)
</Frame>

Scroll down to the **API Key** panel, then select **Generate.**

<Frame>
  ![](/images/rtaImage-820.jpg)
</Frame>

Click on the eye symbol to view your key. You should copy your key and keep it secure.

<Frame>
  ![](/images/rtaImage-821.jpg)
</Frame>

### ****How to re-generate an API key?****

In the same **Integrations** tab.

<Frame>
  ![](/images/rtaImage-822.jpg)
</Frame>

Scroll down to the **API Key** panel, and click **Re-Generate API Key.**

<Frame>
  ![](/images/rtaImage-823.jpg)
</Frame>

Then click **OK** to generate a new API key.

<Frame>
  ![](/images/rtaImage-824.jpg)
</Frame>
