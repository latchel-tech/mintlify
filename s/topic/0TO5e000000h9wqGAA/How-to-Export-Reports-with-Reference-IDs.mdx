### **Appfolio**

If you are using the **Appfolio along with Latchel Chrome Plugin Extension,** follow these steps on how you can export your property and tenant directory from Appfolio.

1. First, **login** to your property management software. Go to **Reports**, scroll to **Tenant Reports**, then click **Tenant Directory**.

   <Tip>
     Alternatively, you can simply search **“Tenant Directory”** using the **search bar** at the top of the page.
   </Tip>
2. In the **Customize Report** window, set **Tenant Status** and **Tenant Type** to **All**, and make sure to **show Active tenants only**. Then, click **Update**.
3. On the right side of the page, **uncheck all options** first. Then **manually select** the following. Make sure that the **rows are not grouped** under any category or information.
   - Property Street Address 1
   - Property Street Address 2
   - Property City
   - Property State
   - Property Zip
   - First Name
   - Last Name
   - Phone Numbers
   - Emails
   - Property ID
   - Unit ID
   - Tenant ID
   - Status (optional)
4. Next, click **Action** at the upper-right corner of the page, then choose **Export as CSV** or **Export as Excel**. **Save** the file to your computer when prompted.
5. Finally, **attach the file to email and send** it to Latchel.

### **Propertyware**

**A. Residents**

Go to **Reports**. If you have existing report folders for **Contacts**, expand **Contact Reports**. Otherwise you can click **New Report** to create report for **Contacts**. Then click **View Report**.

Expand **Contact: Summary**. Make sure the **Entity ID** is selected. Also, check the boxes of the following:

- Full Name
- Email
- Mobile #
- Address
- Address Cont.
- City
- State
- Zip

<Tip>
  **Entity ID** are your tenant's reference IDs which Latchel uses to identify tenants within the integration.
</Tip>

**B. Properties**

Go to **Reports**. If you have existing **Property Reports** within your report folders, **expand it** then select **Units**. Otherwise, click **New Report** at the top to create new report for Units.

<Frame>
  ![1propertyware On Demand Real Estate Solution 12 08 2025 04 42 AM Pn](/images/1Propertyware-On-Demand-Real-Estate-Solution-12-08-2025_04_42_AM.png)
</Frame>

<Frame>
  ![2ropertyware On Demand Real Estate Solution 12 08 2025 05 17 AM Pn](/images/2ropertyware-On-Demand-Real-Estate-Solution-12-08-2025_05_17_AM.png)
</Frame>

Expand **Building Summary**, and make sure to check **Building Entity ID** **_<u>only</u>_** within this section.

<Tip>
  Building Entity ID is your property reference ID which Latchel uses to identify the property address with the integration.
</Tip>

<Frame>
  ![3propertyware On Demand Real Estate Solution 12 08 2025 05 20 AM Pn](/images/3Propertyware-On-Demand-Real-Estate-Solution-12-08-2025_05_20_AM.png)
</Frame>

Next, expand **Unit Summary**. Then, check only the following:

- **Unit Entity ID**
- **Unit Address**
- **Unit Address Cont.**
- **Unit City**
- **Unit State**
- **Unit Zip**
